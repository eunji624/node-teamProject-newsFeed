## 주요 기능

- **반려동물 뉴스피드**
- **로그인/회원가입**: User는 로그인/회원가입을 할 수 있습니다
- **게시판 CRUD**: 로그인한 User는 자신의 반려동물의 사진을 게시판 및 카테고리를 통해 자랑 및 소통 을 할 수있는 공간입니다.
- **댓글 CRUD**: User는 각 게시판에 대해 댓글을 통해 다른 사람들간에 소통을 할 수있는 공간입니다.
